最小生成树（MST）

最小生成树是一个无向图的一个子集，包含所有顶点，且使得边的总权重最小。

适用条件

使用最小生成树算法的条件如下：

  • 图是连通的：所有顶点之间必须有路径相连。
  • 边的权重可以比较：每条边都有一个权重（成本），且可以进行大小比较。
  • 无负权边：边的权重应为非负值，以确保算法正确性。
  • 需要最小成本连接所有顶点：问题需要找到连接所有顶点的最小边权重和。

时间和空间复杂度

Kruskal 算法：

  • 时间复杂度：O(E log E)，E 为边的数量。
  • 空间复杂度：O(V + E)，V 为顶点数量。

Prim 算法：

  • 时间复杂度：O(E log V)（使用优先队列），O(V²)（使用邻接矩阵）。
  • 空间复杂度：O(V)。

算法思路

最小生成树是一个连通图的子图，包含所有顶点且边的总权重最小。以下是两种主要的算法：

  • Kruskal 算法：

    步骤如下：

    1. 首先，列出图中所有的边，并根据边的权重（即成本）从小到大排序。
    2. 接着，从权重最小的边开始，逐条检查这些边。对于每一条边，判断加入这条边后是否会形成环路（即是否会连接两个已经相连的顶点）。
    3. 如果不形成环，就选择这条边，并将其加入最小生成树中。
    4. 重复以上步骤，直到最小生成树包含所有顶点。
  • Prim 算法：

    步骤如下：

    1. 从图中的任意一个顶点开始，将其标记为已访问。
    2. 从已访问的顶点出发，寻找连接未访问顶点的边中，权重最小的那条边。
    3. 选择这条边并将连接的未访问顶点标记为已访问。
    4. 重复这个过程，直到所有顶点都被访问。

using namespace std;

struct Edge {
    int u, v, weight;
};

bool compare(Edge a, Edge b) {
    return a.weight < b.weight;
}

int find(int parent[], int x) {
    if (parent[x] != x)
        parent[x] = find(parent, parent[x]);
    return parent[x];
}

void unionSet(int parent[], int rank[], int u, int v) {
    int rootU = find(parent, u);
    int rootV = find(parent, v);
    if (rootU != rootV) {
        if (rank[rootU] > rank[rootV]) {
            parent[rootV] = rootU;
        } else {
            parent[rootU] = rootV;
            if (rank[rootU] == rank[rootV])
                rank[rootV]++;
        }
    }
}

int main() {
    int V, E;
    cin >> V >> E;
    vector edges(E);
    for (int i = 0; i < E; i++) {
        cin >> edges[i].u >> edges[i].v >> edges[i].weight;
    }
    
    sort(edges.begin(), edges.end(), compare);
    int parent[V], rank[V] = {0};
    for (int i = 0; i < V; i++) {
        parent[i] = i;
    }

    vector mst;
    for (Edge edge : edges) {
        if (find(parent, edge.u) != find(parent, edge.v)) {
            unionSet(parent, rank, edge.u, edge.v);
            mst.push_back(edge);
        }
    }

    // 输出最小生成树
    for (Edge edge : mst) {
        cout << edge.u << " -- " << edge.v << " = " << edge.weight << endl;
    }
}

        cin >> u >> v >> w;
        graph[u].push_back({v, w});
        graph[v].push_back({u, w});
    }

    vector minWeight(V, INF);
    vector inMST(V, false);
    priority_queue, vector>, greater>> pq;

    minWeight[0] = 0; // 从第一个顶点开始
    pq.push({0, 0}); // 权重, 顶点

    while (!pq.empty()) {
        int u = pq.top().second;
        pq.pop();
        inMST[u] = true;

        for (auto &edge : graph[u]) {
            int v = edge.first, weight = edge.second;
            if (!inMST[v] && weight < minWeight[v]) {
                minWeight[v] = weight;
                pq.push({weight, v});
            }
        }
    }

    // 输出最小生成树的边权重
    for (int i = 1; i < V; i++) {
        cout << "0 -- " << i << " = " << minWeight[i] << endl;
    }
}
